# Laundry Wallah / Laundry Mart Webpage

## Project Description
This project is a simple HTML webpage created for the TuteDude HTML task.  
It represents an online webpage for a laundry service called **Laundry Wallah / Laundry Mart**.

The webpage provides basic information about the laundry service, including services offered and an introduction to the business.

## Features
- Header section with welcome message
- About section describing the laundry service
- Image section displaying a laundry-related image
- Services section listing the available laundry services
- Simple and clean HTML structure

## Technologies Used
- HTML5

## Services Provided
- Washing and Drying
- Ironing and Folding
- Dry Cleaning
- Stain Removal
- Express Laundry Service
